Redesing af website - Butik Handler

I dette tema lød opgaven på, at vi i grupper skulle finde en virksomhed, som vi skulle lave et redesign på af deres website. Til dette skulle vi også lave en virksomhedsvideo der skulle kunne bruges på deres nye website, men som også skulle kunne bruges til visning sociale medier.

Processen

Vi arbejdede altså med tre forskellige former for indhold: video- og billederedigering samt kodning og design af website. Vi blev introduceret til grundlæggende fagbegreber indenfor video og billedproduktion og brugte derudover stort set alle de færdigheder vi havde fået fra tidligere temaer:

  • Moodboards
  • Styletile
  • Wireframe
  • Sitemap
  • Layoutdiagrammer
  • Trelloboard
  • Kodning med html, css og javascript

Derudover blev vi også introduceret til github og lærte, hvordan man kunne arbejde i gruppe om den samme kode ved hjælp af github. Yderligere skulle vi både teste brugerne på det gamle website og på vores redesign. Dette gjorde vi ved brug af tre forskellige teste: 5-sek-test, BERT testen og eksperttest. Det skulle nemlig være med til at vise, hvilken brugeroplevelse brugerne havde med det første website og hvilken vej man skulle gå med redesignet til hvordan brugeroplevelsen blev efter redesignet.

Designdelen:

Efter vores første besøg hos Handler, valgte vi at gå med et moderne og overskueligt design, hvor selve billederne og produkterne skulle tale for sig selv, mens baggrunden forblev neutral med sorte fonte og hvid baggrund. Det var nemlig vigtigt for butikken, at der blev lagt vægt på deres kvalitet af varene samt deres ekspertise indenfor faget. Endvidere var det vigtigt for butikken, at de stadig bevaret deres oldschool-udtryk på deres logo, ved at bruge fonten "Minion Variable concept".

Test-indsigter:

Ud fra særligt 5 sekunderstesten var det tydeligt. at ingen kunne se, hvad sitet handlede om før redesignet. Dog efter redesignet var der ingen tvivl om, at brugeren synes sitet handlet om noget indenfor syning, hvilket også var det vi ønskede at fremvise. BERT testen derimod viste resultater af et billigt og simpelt website, som ikke levet op til det æstetiske udtryk, men derimod mere handlede om funktionalitet. Efter vores redesign synes brugeren, at der var lagt mere fokus på æstetisk og udtrykket var mere moderne og var nu heller ikke kvalificeret som billigt, men derimod midtimellem. Dog er vores teste kun testet på enkelte brugere og kunne måske derfor havde givet nogle lidt andre resultater, hvis man testet 100 mennesker.
